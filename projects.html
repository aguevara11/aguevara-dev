LEAN - Family Fitness Managment Web Application

In Progress

LEAN is a family fitness management application that helps families set and achieve health goals together. Each member has an account to track key health metrics like steps, sleep, and activity levels, contributing to shared family progress. Families can compete in challenges against others, fostering motivation and accountability.

The platform integrates AI for personalized insights, SQL databases for secure data storage, Firebase for authentication, and APIs for seamless connectivity. Built with HTML, JavaScript, and CSS, LEAN makes fitness a fun and collaborative experience.

Mario Bird - Flappy Bird Replica

2024

Mario Bird is a side-scrolling game I developed, inspired by Flappy Bird but with a Mario-themed twist. I designed the game graphics myself and built the game using HTML, CSS, and JavaScript. The project showcases my skills in game development, animation, and interactive design.

This project was completed in my Game Design class, where I gained hands-on experience with character physics, animation, and interactive mechanics. I learned how to apply physics engines to create realistic movements and how to design engaging gameplay elements that enhance the player experience.
